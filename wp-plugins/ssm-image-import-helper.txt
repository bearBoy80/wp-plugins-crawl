You will need to add an alt tag manually.<\/p>\n<p>With the plugin turned on, the image would be saved as &#8220;berlin-wall.jpg&#8221; (url compliant), and the title tag would be &#8220;berlin wall&#8221; and an alt tag would automatically be created, &#8220;berlin wall.&#8221;<\/p>\n<p>Here&#8217;s another example &#8211; we import a file named &#8220;Andy Warhol&#8217;s Marilyn&#8221;. The title tag is &#8220;Andy Warhol&#8217;s_Marilyn&#8221; (again, not very searchable).<\/p>\n<p>With the plugin turned on, the file is saved as &#8220;andy-warhols-marilyn.jpg&#8221;, with a title &#8220;Andy Warhol&#8217;s Marilyn&#8221; and an alt tag of &#8220;Andy Warhol&#8217;s Marilyn.&#8221;<\/p>\n<p>So, for best results, name your images using capital letters where appropriate (eg, place or people names), and separate words using either spaces, hyphens or underscores.<\/p>\n<p>Please note the plugin is not (yet) able to read minds, so if your original file is named
