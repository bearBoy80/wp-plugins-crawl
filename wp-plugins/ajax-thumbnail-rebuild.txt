This works fine when you don\u2019t have that many photos on your site. When you have a lot of full-size photos, the script on the server side takes a long time to run. Unfortunately the time a script is allowed to run is limited, which sets an upper limit to the number of thumbnails you can regenerate. This number depends on the server configuration and the computing power your server has available. Your old thumbnails won\u2019t be resized. That\u2019s where this plugin comes into action. After changing the image sizes, you can rebuild all thumbnails. But instead of telling the server to recreate all thumbnails at once, they are rebuilt one after another. Rebuilding thumbnails for one photo won\u2019t take all too long, so you won\u2019t run into any script timeouts. Note that you still have to wait until all thumbnails have been rebuilt.
